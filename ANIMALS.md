# ANIMALS.md — 动物设计规格（Animal Design Spec）

> 本文件是 [DESIGN.md](./DESIGN.md) 第8节 Animal Perspective 的可执行规格版本，服务于 [EVENTS.md](./EVENTS.md) 的事件写作，以及 `index.html` 里 Idle Narrative / Today's Memory 的内容设计。
>
> **这不是动物百科介绍，而是游戏设计文档**——每一条内容都直接回答"这个行为在游戏里应该落进哪个系统（事件 / Idle / Today's Memory）、应该以什么频率出现"，而不是单纯描述动物本身的知识。三种动物的行为清单里如果出现重叠（比如都会"竖耳朵"），是刻意的——真正拉开差异的不是"有没有这个行为"，而是这份文档接下来定义的**反应节奏、触发方式、和出现频率**。

## 设计取舍原则（同一行为有多种呈现方式时，优先选哪种）

当一个动物行为可以有多种写法时，本文件统一优先选择更符合"第一人称沉浸体验"的方案，具体是这三条：

1. **优先"被动反应"而非"主动规划"**：动物应该是在回应一个具体的世界层刺激（声音/气味/动作），而不是"决定要去做某件事"——后者是人类视角的安排式思维，正是 Design Motivation 里想避开的东西。例如"狗想出去散步"不成立，"狗看见主人拿起遛狗绳，身体先兴奋起来"才成立。
2. **优先"身体先反应，意识后跟上"而非相反**：尤其是兔子这类被捕食动物，反射速度应明显快于"思考"，这样才符合真实的动物本能，也最容易让玩家感觉"我是这只动物"而不是"我在操控这只动物"。
3. **优先具体感官线索，而非抽象心理描写**：每个行为都应该能落到"这只动物的哪个感官先注意到了什么"，呼应第7节 Event Layer 的世界层→感知层结构。

## 猫

### 1. 核心性格
独立、好奇、谨慎——这三者会互相拉扯：好奇心把猫推向新奇的事物，谨慎又让它不会贸然行动，于是"观察很久才决定要不要靠近"是猫最典型的决策节奏。这种"想靠近又克制"的张力，是猫专属事件在选择设计上应该体现的核心性状，"现在就确认 / 再观察一会儿 / 干脆不管"的三段式犹豫（呼应已实现的 EXP-003「窗边的小鸟」）是这个性状的一种具体表达方式，不是唯一允许的表达方式。**这条描述的是猫的核心性状，不代表每个猫事件都必须套用这个具体的三段式选项结构**——如果 DESIGN.md 第14.5节 Experience Pattern Audit 显示这个 Pattern（本质上属于"接近 vs 保持距离"这一类）已经过量，应该换一种 Pattern（如"等待 vs 行动""接受 vs 拒绝"）来表达同一种犹豫性格，而不是重复同一个选项骨架。

### 2. 感官特点
听觉最敏锐，动态视觉（对移动物体）也很敏锐，夜视很好，嗅觉是三种动物里相对最弱的（但仍远超人类）。

**设计含义**：猫的感知层文案应该以"听见 / 看见移动"为主线索，"闻到"为辅——如果某个世界层事件本来是"闻到气味"触发，落到猫身上时应弱化嗅觉描写，改写成"注意到了动静""耳朵先有反应"，保持感官真实感。

### 3. 常见行为
趴窗台、晒太阳、舔毛 / 理毛、踩奶、盯着移动的东西看、突然炸毛后又若无其事、找高处、对着看不见的地方发呆（其实在听声音）。

### 4. 可作为事件的习性
- 看到移动的东西（虫子 / 光斑 / 逗猫棒）→ 扑 / 观察 / 无视
- 听到不明声响 → 靠近查看 / 继续观察 / 躲起来
- 陌生气味或新物体（纸箱 / 客人的包）→ 钻进去 / 绕圈闻 / 保持距离
- 高处诱惑（柜子顶 / 窗台）→ 跳上去 / 留在原地
- 主人的动作（开零食袋 / 拿罐头）→ 上前蹭 / 叫 / 无动于衷

这些事件的世界层触发都应该是"外部刺激先出现"，而不是"猫决定要做什么"——猫是被动反应者，不是主动规划者。

### 5. 可作为 Idle Narrative 的习性
舔毛、打哈欠、踩奶、慢慢甩尾巴、耳朵转向某个方向又收回、眯眼晒太阳、盯着虚空发呆、换个姿势趴好。共同点：没有明确的"决策后果"，只传达"猫还活着、还在感受这个世界"，适合填充事件之间的空档。

### 6. 可作为 Today's Memory 的行为
- 第一次主动靠近陌生事物后确认"安全"（好奇心被验证）
- 罕见地钻进了很隐蔽的地方（柜子、纸箱深处）
- 对声音的一次强烈反应（炸毛 / 迅速躲藏），事后回忆起来是"有惊无险"的片段
- 主人回家时"没有立刻扑过去，只是慢慢走过去蹭了一下"（呼应 OWN-001，独立又亲近的矛盾感是猫最具代表性的记忆时刻）

### 7. 高频行为
晒太阳、舔毛、盯着移动物体看、对声响有反应——建议 Common 稀有度，几乎每天都出现，构成猫日常的基本节奏。

### 8. 稀有行为
钻进很少去的角落（衣柜 / 纸箱深处）、对猫薄荷失控地打滚、罕见地主动坐到主人腿上——建议 Rare / Secret 稀有度，作为"这一天不一样"的信号。

### ① Behaviour Rules（行为频率）

| 频率 | 行为 |
|---|---|
| 高频（约 70%） | 舔毛 / 理毛、晒太阳、打哈欠、换姿势趴好、耳朵转向声音又收回、眯眼放松、对突然声响的短暂警觉反应、踩奶（放松状态下） |
| 中频（约 25%） | 跳上高处（柜子 / 窗台）、对新物体（纸箱 / 客人的包）绕圈闻、追逐移动的东西（光斑 / 虫子 / 逗猫棒）、听到零食声响上前蹭 / 叫、炸毛后恢复平静 |
| 稀有（约 5%） | 钻进很隐蔽的角落（衣柜 / 纸箱深处）、对猫薄荷失控打滚、罕见地主动坐到主人腿上、长时间维持某种紧密互动而不走开 |

### ② Trigger Rules（触发规则）

| 行为 | 时间段 | 关键状态条件 | 环境条件 | 前置事件 | 说明 |
|---|---|---|---|---|---|
| 扑向移动物体 / 玩具 | 不限（白天更常见） | 好奇较高、疲劳不太高 | 室内 | 无 | 纯反射型，几乎不需要门槛，高频 |
| 靠近 / 查看不明声响 | 不限 | 紧张不太高（否则倾向躲） | 室内 | 无 | 谨慎但仍会探究，中频偏高频 |
| 钻进新物体（纸箱 / 客人的包） | 不限 | Hunger 不高（有闲心探索）、紧张较低 | 室内 | 出现新物体这一世界层事件 | 好奇心驱动，中频 |
| 跳上高处（柜子 / 窗台） | 白天 / 傍晚更常见 | 不限 | 室内 | 无 | 展示猫对高处的偏好，中频 |
| 因零食声响上前蹭 / 叫 | 三餐前后 | Hunger 较高 | 主人在家 | 无 | 唯一明确需要"主人在场"的高频候选行为 |
| 主人回家蹭一下（而非扑过去） | 傍晚 | Bond 达到一定水平 | 主人刚回家 | "主人回家"世界层事件刚发生 | Today's Memory 高光候选，呼应 OWN-001 |
| 钻进极隐蔽角落 / 对猫薄荷打滚 | 不限 | 情绪层.放松很高、紧张很低 | 室内、无外部打扰 | 当天此前没有被紧张类事件打断 | 稀有，代表这一天整体很安心 |

> ⚠️ 表格里"主人在家"这类条件，目前的状态系统（DESIGN.md 第6节）还没有对应的显式变量，需要像 EVENTS.md 的"天气"字段一样，作为后续实现时的新增前置需求记录下来，而不是凭空假设已经存在。

### ③ Never Rules（行为边界）

- 不应该"立刻扑向主人讨摸"这种狗式的热情行为——猫的亲近应该是"蹭一下就走开"，而不是持续黏人。
- 不应该表现出"很容易被陌生人喜欢 / 靠近"——即使 Bond 很高，猫对陌生人也应默认谨慎，不能轻易表现出信任。
- 不应该"经常大声喵叫求关注"——猫更倾向于安静观察，叫声应保留给真正的高频需求（如饥饿），不能滥用为通用的"求互动"手段。
- 不应该表现出"合作式的服从行为"（比如"听主人指令做动作"）——这是狗的行为逻辑，不该出现在猫身上。
- 不应该频繁"主动追逐陌生的狗 / 大型动静"——猫的好奇心止步于安全距离内，不应该表现得像狗一样莽撞。

### ④ Daily Rhythm（一天的行为节奏）

猫是晨昏活动型动物：一天不是"事件均匀分布"，而是两头活跃、中间大段沉睡的形状。

| 时段 | 主导基调 | 高发行为类别 | 事件密度 | 呼应的已有事件 |
|---|---|---|---|---|
| Morning（07:00-11:00） | 好奇 + 期待 | 苏醒后的短暂活跃期，容易被移动物体吸引，也是进食本能最强的时段 | 中 | EAT-001、EXP-003、PLY-001 |
| Noon（11:00-15:00） | 慵懒 + 平静 | 全天最沉的一段——晒太阳、理毛、打盹，几乎不该主动触发探索类事件 | **低**（idle 占比应最高） | RST-001、RST-002 |
| Evening（15:00-20:00） | 从慵懒切换到警觉 + 期待 | 第二个活跃高峰，主人回家相关的社会层事件集中在这里，也是光影变化类事件的黄金时段 | 高 | OWN-001、EXP-002 |
| Night（20:00-24:00） | 平静，偶有惊喜 | 大部分时间放松，但保留夜行本能偶尔浮现的空间 | 低，但含稀有内容 | RAR-003 |

**设计意图**：猫的一天应该呈现"两头尖、中间平"的形状——Noon 时段刻意压低事件密度，是唯一能让"猫大部分时间在睡觉"这条 ANIMALS.md 核心性格真正被玩家感受到的地方；如果 Noon 时段事件照样密集，猫就会显得和狗一样"随时精力充沛"，失去物种差异。

## 狗

### 1. 核心性格
热情、忠诚、社交欲望强，行动力高。狗的决策节奏和猫相反：几乎不犹豫，感知到刺激后会立刻用身体做出反应（叫、冲、摇尾巴）。狗的事件选项不该做"要不要观察一下"这种迟疑型分支，而应该是"用什么方式回应"的分支（例如：叫出声 / 用身体贴近 / 绕着对象转圈）——保留"马上有反应"这个特质，只是反应形式不同。

### 2. 感官特点
嗅觉是三种动物里最强的，听觉也很好，夜视一般。

**设计含义**：狗的感知层文案应该以"闻到"为主线索来源——很多对猫来说是"听到声音"的世界层事件，落到狗身上时更应该改写成"鼻子先有反应"，符合狗以嗅觉为主导感官的真实设定。

### 3. 常见行为
闻空气 / 闻地面、等在门口、摇尾巴、追球 / 捡东西、看到绳子 / 外套就兴奋、主人回来时冲过去、对着窗外经过的人或狗叫、翻肚皮讨摸。

### 4. 可作为事件的习性
- 闻到食物 / 陌生气味 → 追过去查看 / 叫出声表达兴趣 / 忽略
- 听到 / 看到"要出门的信号"（拿绳子、拿钥匙、穿鞋）→ 兴奋地围过去 / 叼东西示意 / 原地等待
- 陌生人或陌生狗经过（门外 / 窗外）→ 冲上前警戒 / 隔着距离叫 / 好奇张望
- 主人情绪变化（如说话声音不对 / 在哭）→ 贴上去蹭 / 安静守在旁边
- 球 / 飞盘一类可捡拾的东西出现 → 立刻叼回来 / 围着转圈玩

狗的事件应尽量让"世界层信号"具体而明确（绳子、钥匙、气味），因为狗对这些具体信号的条件反射式反应，本身就是最容易让玩家"感觉自己是一只狗"的地方——比抽象的心理活动更有代入感。

### 5. 可作为 Idle Narrative 的习性
甩耳朵、打哈欠、舔鼻子、摇尾巴、转圈准备趴下、闻自己的爪子、耳朵突然竖起又放下（听到远处声音但没到需要反应的程度）。

### 6. 可作为 Today's Memory 的行为
- 主人回家时冲过去迎接的瞬间（狗最典型的"高光时刻"来源，是 Relief / Happiness 情绪标签的主力）
- 一次特别投入的捡球 / 追逐
- 陪着情绪低落的主人安静待了很久
- 因为陌生人经过而叫了很久，后来确认没事、恢复平静

### 7. 高频行为
摇尾巴、闻气味、对声响有反应、等主人——建议 Common 稀有度，构成狗日常最基本的存在感。

### 8. 稀有行为
主人难得一整天没出门（对应 EVENTS.md RAR-002「主人今天请假」）、遇到一只印象深刻的陌生狗、得到额外的零食奖励——建议 Rare / Secret 稀有度，是"今天特别开心/特别"的信号。

### ① Behaviour Rules（行为频率）

| 频率 | 行为 |
|---|---|
| 高频（约 70%） | 摇尾巴、闻空气 / 闻地面、打哈欠、舔鼻子、耳朵对声音的常规反应、等在门口 / 等主人、转圈准备趴下 |
| 中频（约 25%） | 追球 / 捡东西、对着窗外经过的人 / 狗叫、看到出门信号（绳子 / 钥匙）兴奋围过去、陌生人经过时冲上前 / 隔距离叫、翻肚皮讨摸 |
| 稀有（约 5%） | 主人一整天没出门、全程跟随主人、陪伴情绪低落的主人很久、遇到印象深刻的陌生狗、得到额外零食奖励 |

### ② Trigger Rules（触发规则）

| 行为 | 时间段 | 关键状态条件 | 环境条件 | 前置事件 | 说明 |
|---|---|---|---|---|---|
| 摇尾巴 / 闻气味（基础反应） | 不限 | 不限 | 室内 | 无 | 高频，几乎伴随所有互动，是狗的"默认状态" |
| 追球 / 捡东西 | 白天更常见 | 疲劳不高 | 有可捡拾物体出现 | 玩具 / 球出现这一世界层事件 | 中频 |
| 看到出门信号（绳子 / 钥匙）兴奋围过去 | 不限，早晚更常见 | 不限 | 主人在家且做出准备出门的动作 | "主人拿起绳子 / 钥匙"世界层事件 | 需要新增"即将出门"这类世界层信号 |
| 陌生人 / 狗经过时叫 / 警戒 | 不限 | 紧张略升 | 门外 / 窗外有动静 | 无 | 中频 |
| 主人回家冲过去迎接 | 傍晚居多 | Bond 不作硬性门槛（狗天生高羁绊倾向） | "主人回家"事件刚发生 | "主人回家"世界层事件 | 高频且稳定的 Today's Memory 来源；与猫的"蹭一下"形成对照，狗应默认选"冲过去" |
| 陪伴情绪低落的主人很久 | 不限 | Bond 较高 | 主人情绪变化被感知到 | "主人说话声音不对 / 哭"这一世界层事件 | 稀有，重要 Today's Memory 候选 |
| 主人一整天没出门 | 不限 | Bond > 40（EVENTS.md RAR-002 已定义） | 当天没有触发"主人出门"事件 | 无 | 稀有 |

> ⚠️ 表格里"主人在家 / 主人即将出门"这类条件，目前的状态系统还没有对应的显式变量（和猫的表格是同一个待补需求），需要作为后续实现的新增前置需求记录。

### ③ Never Rules（行为边界）

- 不应该表现出"长时间独自观察不行动"——这是猫的节奏，狗几乎总会很快做出身体反应，长时间犹豫不符合狗的性格。
- 不应该"对陌生人 / 陌生动静完全冷漠不理会"——狗的社交天性意味着几乎不会对新鲜事物毫无反应，即使是警戒这种负面反应，也比"完全无视"更符合狗的设定。
- 不应该表现出"很容易独立躲藏很久不出来"——这是兔子的行为逻辑，狗即使紧张也更倾向于靠近主人，而不是躲进角落。
- 不应该"安静地舔毛整理很久"——猫式的自我整理不应该是狗的高频 / 中频行为，狗简单舔一下鼻子 / 爪子即可，不该发展成长时间的理毛场景。
- 不应该表现出对主人"若即若离"——狗的 Bond 增长逻辑应始终是正向、主动的，不应该出现"故意保持距离考验关系"这种猫式的欲擒故纵。

### ④ Daily Rhythm（一天的行为节奏）

狗的一天不是按"物种生理节律"分布，而是紧紧跟着"主人的作息"走——这是狗和猫、兔子最大的结构性差异：猫、兔子的节奏由自己的生理决定，狗的节奏由别人（主人）的行程决定。

| 时段 | 主导基调 | 高发行为类别 | 事件密度 | 呼应的已有事件 |
|---|---|---|---|---|
| Morning（07:00-11:00） | 精力最旺盛 | 对气味、新鲜事物反应最积极；如果有"出门信号"类情境，应优先安排在这里 | 高 | EAT-001、PLY-001 |
| Noon（11:00-15:00） | 等待，轻度焦虑 | 如果主人不在家，这是狗最容易表现"等待"和"轻度不安"的时段——闻空气、听远处声音、等在门口 | 中，情绪基调偏"悬着" | EAT-002 |
| Evening（15:00-20:00） | 全天情绪峰值 | 主人回家事件应几乎必然安排在这里，是狗 Today's Memory 最重要的高光来源 | **高**（全天密度最高的时段） | OWN-001 |
| Night（20:00-24:00） | 逐渐安静，但不"关机" | 依旧保持基础警觉，不应该表现出对声音完全无反应 | 中低 | RST-002 |

**设计意图**：狗和猫在 Noon 时段应该形成鲜明对照——猫在这个时段是"睡觉、密度最低"，狗是"等待、带着悬着的情绪、密度中等但基调不轻松"。这个对比本身就是让两种动物"感觉不一样"最直接的手段，不需要额外机制，只需要在同一个时段安排不同基调的内容。

## 兔子

### 1. 核心性格
胆小、警觉、温柔——兔子的决策节奏是三种动物里最保守的：默认反应是"先假设有危险，退到安全的地方"，只有反复确认安全之后才会放松。兔子的事件选项经常适合包含"逃避 / 躲藏"这个选项，并且这个选项**不应该被塑造成"错误答案"**——躲起来对兔子来说是完全合理、甚至是常态的反应，事件设计不该暗示玩家"应该"选择更勇敢的选项。**但"经常包含躲避选项"不等于"必须套用接近/回避这个 Pattern"**——如果这个 Pattern 已经过量（见 DESIGN.md 第14.5节 Experience Pattern Audit），同样的谨慎性格也可以通过"顺应本能 vs 克制""等待 vs 行动"等其它 Pattern 表达，不必每次都落回"躲不躲"这个具体形状。

### 2. 感官特点
听觉极强（和猫并列三种动物里最高），嗅觉也不错，夜视一般（兔子是晨昏活动型动物，不是纯夜行动物）。

**设计含义**：兔子的感知层文案应该高度依赖"听到"，且反应速度要写得比猫、狗都快——猫是"耳朵动了动"，兔子应该是"耳朵唰地转过去，身体已经先僵住了"，强调本能反射先于意识判断，这是兔子作为被捕食动物最核心的真实感。

### 3. 常见行为
竖耳朵 / 转耳朵、啃咬东西磨牙、缩在角落、突然僵住、快速跳开、试探性地探出头、蹭一蹭表示放松、偶尔的小跳跃（binky，真正放松愉悦时才会出现）。

### 4. 可作为事件的习性
- 突然的声响（关门声 / 物品掉落）→ 立刻躲进最近的隐蔽处 / 僵住不动 / 竖耳朵判断方向
- 陌生的手或陌生人靠近 → 缩得更紧 / 试探性地凑近闻一下 / 逃到更远的地方
- 熟悉的脚步声靠近 → 探出头观察 / 保持警惕不动 / （信任建立后）主动凑近
- 可啃咬的东西出现（纸箱边缘 / 玩具）→ 上前啃咬 / 只是闻一闻 / 不理会
- 长时间的安静环境 → 放松地趴下 / 依然保持警惕姿势

兔子几乎所有事件都应该以"我先感觉到了什么，身体先做出反应"开头，而不是"我注意到了什么，我决定做什么"——反应速度和意识判断之间要拉开明显差距，这是兔子和猫、狗最大的叙事差异点，也是最容易体现"我是一只很容易害怕的小动物"这种第一人称沉浸感的地方。

### 5. 可作为 Idle Narrative 的习性
耳朵轻轻动了动、抽动鼻子、理毛、缩成一团、小跳一下（binky，仅在心情很好时出现）、试探性地探出头又缩回去。

### 6. 可作为 Today's Memory 的行为
- 一次由紧张到放松的完整过程（听到声音躲起来 → 确认安全 → 慢慢恢复）
- 罕见的一次 binky（跳跃），代表"这一天真的很安心"
- 第一次没有躲避、而是主动靠近了熟悉的人
- 长时间维持警惕后终于能安静趴下的瞬间

### 7. 高频行为
竖耳朵、对声音警觉、缩起来观察——这些是兔子最基本的生存本能，建议 Common 稀有度，几乎贯穿全天。

### 8. 稀有行为
binky 跳跃（现实兔子行为学里公认的"高兴信号"，天然适合做 Secret 稀有度）、主动蹭人、长时间毫无防备地放松趴着——建议 Rare / Secret 稀有度，因为对一只天性警惕的动物来说，"完全放松"本身就是最稀有、最值得记住的状态。

### ① Behaviour Rules（行为频率）

| 频率 | 行为 |
|---|---|
| 高频（约 70%） | 竖耳朵 / 转耳朵、抽动鼻子、对声响的警觉反应（僵住 / 竖耳朵判断方向）、啃咬东西磨牙、试探性探出头又缩回、理毛 |
| 中频（约 25%） | 因声响 / 陌生刺激缩进隐蔽处躲避、熟悉脚步声靠近时探出头观察、长时间安静后放松趴下、陌生的手靠近时的反应 |
| 稀有（约 5%） | binky 小跳跃、主动蹭人、长时间毫无防备地放松趴着、第一次没有躲避而主动靠近 |

### ② Trigger Rules（触发规则）

| 行为 | 时间段 | 关键状态条件 | 环境条件 | 前置事件 | 说明 |
|---|---|---|---|---|---|
| 竖耳朵 / 转耳朵（基础警觉） | 不限 | 不限 | 室内 | 无 | 高频，几乎持续存在的"底噪"反应 |
| 啃咬东西磨牙 | 不限 | 紧张不太高 | 有可啃咬物 | 无 | 高频 |
| 突然声响 → 躲进隐蔽处 / 僵住 | 不限，夜间 / 安静时更敏感 | 安全感基线较低时更容易触发强反应 | 室内 | 任意较大声响这一世界层事件 | 高频到中频，取决于声响强度 |
| 陌生手 / 人靠近 → 缩紧或试探 | 不限 | Bond 较低时倾向缩紧，Bond 较高时倾向试探 | 有陌生人 / 手靠近 | 无 | 中频，Bond 直接影响选择倾向 |
| 熟悉脚步声 → 探出头 / 保持警惕 / 主动凑近 | 傍晚居多 | Bond 越高越倾向"主动凑近"而非"保持警惕" | "主人回家"事件刚发生 | "主人回家"世界层事件 | Bond 数值影响选择倾向的典型例子 |
| binky 小跳跃 | 不限，但需长时间无紧张事件 | 情绪层.放松很高、紧张很低、安全感很高 | 室内、长时间无外部打扰 | 之前一段时间没有发生任何紧张类事件 | 稀有，兔子最典型的 Today's Memory 高光 |
| 第一次没有躲避而主动靠近 | 不限 | Bond 首次超过较高阈值 | 无 | 需要 Bond 跨过某个阈值（如 50+） | 稀有，标志关系里程碑 |

> ⚠️ 与猫、狗一样，"主人在家 / 主人回家"这类条件依赖的状态变量目前还不存在，是三种动物共用的同一个待补需求，不是兔子专属问题。

### ③ Never Rules（行为边界）

- 不应该表现出"主动冲向陌生刺激确认"——这是狗的行为逻辑，兔子面对不确定的新鲜事物，默认反应永远是先退、先躲，而不是主动上前。
- 不应该"频繁大幅度移动 / 长时间兴奋乱跑"——兔子的能量表现应该是间歇性的（偶尔一次 binky），而不是持续性的活跃，长时间高能量状态不符合兔子谨慎的天性。
- 不应该表现出"对声音无动于衷"——听觉是兔子最核心的感官，任何声响都应至少引发耳朵反应，完全无反应会破坏兔子的角色设定。
- 不应该"轻易对陌生人表现出信任"——即使 Bond 很高，兔子对"熟悉的人"和"陌生人"的反应差异也应保持明显，不能让兔子变得跟狗一样对谁都热情。
- 不应该"长时间维持同一姿势、毫无警觉动作穿插"——即使在最放松的状态，兔子偶尔的耳朵抽动 / 试探性抬头也应保留，完全静止不动会让兔子显得像别的动物。

### ④ Daily Rhythm（一天的行为节奏）

兔子和猫一样是晨昏活动型，但警觉基线贯穿全天、从不真正"关闭"——这是兔子和猫最大的节奏差异：猫在 Noon 时段可以完全放松沉睡，兔子即使在最安静的时段也要保留少量警觉动作。

| 时段 | 主导基调 | 高发行为类别 | 事件密度 | 呼应的已有事件 |
|---|---|---|---|---|
| Morning（07:00-11:00） | 晨间警觉 | 刚睡醒时最容易被细微变化触发，好奇心和警觉并存 | 中 | EAT-001、EXP-003 |
| Noon（11:00-15:00） | 相对安静，但不完全放松 | 理毛、晒太阳，是全天最容易维持平静的时段，但仍应保留少量"竖耳朵"类底噪反应 | 低，但不为零 | RST-001 |
| Evening（15:00-20:00） | 第二个警觉高峰 | 熟悉脚步声事件的黄金时段，Bond 高时更倾向"主动凑近"而不是"保持警惕" | 中高 | OWN-001、EXP-002 |
| Night（20:00-24:00） | 情绪弧线的终章 | 最容易触发"完全放松"类稀有时刻（如 binky）的时段，需要前面长时间无紧张事件累积 | 低，但含全天最珍贵的内容 | RAR-003 |

**设计意图**：兔子和猫在 Noon 时段的差异必须体现出来——猫可以"事件密度降到零、纯睡觉"，兔子即使在最安静的时段也不能完全没有警觉细节（呼应 ③ Never Rules"不应该长时间维持同一姿势毫无警觉动作"）。兔子全天真正的情绪高点不在白天，而在 Night 时段"确认了一整天都安全"之后才可能出现——这也是为什么 binky 应该被安排在 Night 而不是随便什么时候。

## 三种动物的差异速查表

| 维度 | 猫 | 狗 | 兔子 |
|---|---|---|---|
| 决策节奏 | 先观察，犹豫后再行动 | 几乎不犹豫，立刻反应 | 本能先动，意识后跟上 |
| 主导感官 | 听觉 + 动态视觉 | 嗅觉 | 听觉 |
| 事件选项设计倾向（性状参考，非固定结构，见各自①核心性格的说明） | 犹豫、观察后再决定 | "用什么方式回应"的反应分支 | 谨慎，躲避是合理选项而非错误答案 |
| 稀有行为的意义 | 罕见的深度探索 / 亲近 | 罕见的额外惊喜（陪伴/奖励） | 罕见的"完全放松"状态 |
| Today's Memory 常见基调 | 好奇 / 独自观察 | 羁绊 / 热闹相伴 | 安全感 / 安静躲藏 |
